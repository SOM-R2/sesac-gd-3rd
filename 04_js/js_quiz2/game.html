숫자 Up and Down 게임

[게임 시작하기] 버튼을 클릭하면 startGame()함수가 실행(호출)되도록 미리 설정했습니다. 아래 조건을 만족하도록 startGame()함수를 내부 코드를 채워주세요.
  • prompt()함수와 alert()를 이용해 숫자 Up and Down 게임을 만들어볼까요?
  • 정답 숫자를 저장하는 변수 answer의 값은 Math.random() 메소드를 이용해서 0 < x <= 10 까지의 수 로 설정해주세요
  • while 반복문은 계속해서 반복됩니다.
  • 변수 guessValue에는 prompt()함수를 이용해 사용자가 추측한 값이 저장됩니다.
    let guessValue = prompt('UP and DOWN 게임을 시작합니다!');라고 코드를 작성하면 프롬프트에 사용자가 입력한 값이 guessValue 변수에 "문자열"로 저장됩니다.
  • 만약에 사용자가 추측한 값이 "q" 또는 "포기"였다면, 게임이 종료되고 alert()함수로 "포기하셨습니다.." 창이 나타납니다. 그리고 반복이 종료됩니다.
  • 만약에 사용자가 추측한 값이 정답과 같다면, alert()함수로 (만약에 random()수가 10이었다면) "10!! 정답입니다!" 창이 나타납니다. 그리고 반복이 종료됩니다.
  • 만약에 사용자가 추측한 값이 정답보다 크다면, alert()함수로 "Down" 창이 나타납니다. alert창의 [확인] 버튼을 누르면 다음 반복으로 넘어가 prompt 대화창이 나타납니다.
  • 만약에 사용자가 추측한 값이 정답보다 작다면, alert()함수로 "Up" 창이 나타납니다. alert창의 [확인] 버튼을 누르면 다음 반복으로 넘어가 prompt 대화창이 나타납니다.
